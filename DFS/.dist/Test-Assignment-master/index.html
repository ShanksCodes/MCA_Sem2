Programming is like any other sport. You might know the rules, but you have to play to learn - Mark Zuckerberg

Boosting the concept of Programming we bring forward a unique coding challenge. Blind coding is the new platform where each and every one of you has a chance to win.

With the idea of imparting programming knowledge, Presenting Blind Coding, a simple coding competition where you'll be given even simpler problems to code. Sounds too easy? It isn't! The catch is that you will be forbidden to look at your code!

Event Specification

  1. This is an Individual Event.
  2. This code can be written using VSCode or Notepad.
  3. This event consists of two rounds.
  4. In Round-1 HTML CSS template of medium complexity will be given.
  5. In Round-2 JS program statement of complexity will be given.
    6. ****Participants have to write code with monitors off.

Rules

  1. ****Each participant has to write the code for the problem statement, with MONITOR OFF.
  2. Participants whose produces similar design and gets the code executed wiil be given higher
  3. The second preference will be given to code with least number of errors.
  4. In case of tie in number of errors then timing will be considered.
  5. Shortlisted participants of Round I will appear for Round II, for which all the above rules
